### üìä Logbook Feed
- View your complete climbing history in a clean, organized format
- Sort by grade or date (ascending/descending)
- Filter by grade, name, or number of attempts
- Track your climbing journey over time

## üöÄ Getting Started

### Prerequisites
- Node.js (v18 or later recommended)
- npm or yarn
- Kilterboard account

### Installation

1. Clone this repository
```bash
git clone https://github.com/yourusername/kilterboard-wrapped.git
cd kilterboard-wrapped
```

2. Install dependencies
```bash
npm install
# or
yarn install
```

3. Run the development server
```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser
